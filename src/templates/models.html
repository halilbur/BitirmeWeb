ResNet18 Architecture

Model Specifications

  • Architecture: Residual Network with 18 layers
  • Input Size: 512x512x3 RGB images
  • Output Classes: 12 clothing categories
  • Parameters: 11,445,324 trainable parameters
  • Training Data: 5995 clothing images
  • Validation Data: 664 5995 clothing images
  • Test Data: 1667 clothing images
  • Training Epochs: 200 epochs
  • Batch Size: 32 images per batch
  • Loss Function: Cross-entropy loss
  • Optimizer: Adam with learning rate 0.001

Key Features

Skip Connections: Prevents vanishing gradient problem

Transfer Learning: Pre-trained on ImageNet

Data Augmentation: Resize, Horizontal Flip, Vertical Flip, Rotation, Color Jitter, Sharpen, ToTensor

Prototypical Networks

Model Specifications

  • Architecture: Few-shot learning with prototypes
  • Embedding Dimension: 512-dimensional feature space
  • Distance Metric: Euclidean distance
  • Support Set: 5-way 1-shot learning
  • Episodes: 10,000 training episodes
  • Backbone: 4-layer CNN encoder

Key Features

Few-Shot Learning: Works with limited examples

Prototype Centers: Learns class representations

Episodic Training: Mimics test conditions

Style Similarity: Captures fine-grained patterns

Triplet Loss + XGBoost

Model Specifications

  • Triplet Network: Siamese architecture with triplet loss
  • Margin: α = 0.2 for triplet loss
  • Embedding Size: 256-dimensional vectors
  • XGBoost Trees: 100 estimators
  • Max Depth: 6 levels per tree
  • Learning Rate: 0.1 for gradient boosting

Key Features

Triplet Loss: Learns similarity embeddings

Hard Mining: Focuses on difficult examples

Gradient Boosting: Ensemble learning approach

Feature Engineering: Hand-crafted + learned features
